# zfsgui
MacOS widget for ZFS file system

### Building the app:

You will need to download sources for the following libraries:
- openzfsonosx/zfs

and install the following python dependencies:

- Cython (from pip)
- rumps (from pip)
- watchdog (from pip)
- py2app (from pip)
- rejsmont/py-libzfs (after #72 PR, freenas/py-libzfs)
